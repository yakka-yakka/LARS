# LARS: Legion Arma Reforger Suite

LARS is an all-in-one background overlay service tightly integrated with Arma Reforger's engine. It seamlessly injects advanced dynamic Hotbar Syncing profiles and a robust undetectable Anti-AFK looping sequence directly into your game via precise AutoHotkey injection calls underneath a sleek native Python `PyQt5` click-through interface.

---

### Features
1. **Dynamic Hotbar Profile System (F11)**: Build multiple loadout profiles in your `settings.ini` out-of-game or on the fly. 
2. **Real-time Hotbar Recording (Hold F9)**: Use your mouse inside the game to programmatically map and register new hotkey grid coordinates into the LARS UI memory effortlessly.
3. **Automated Inventory Syncing (Tap F9)**: LARS automatically bypasses manual drags and micro-glides all tracked inventory items straight into your Arma hotbar with precisely mapped execution timings to bypass rendering stutters.
4. **Built-in Anti-AFK Heartbeat (F10)**: Keeps server connections active by executing randomized sweeping mouse gestures, variable WASD key taps, and random combat reloads safely inside the engine on a perfectly disguised intermittent 30-second timer.
5. **Zero Dependencies Installation**: Utterly portable setup routines dynamically fetch and build completely localized zero-install dependencies out of `python.exe` and `AHK v2`.

---

### Installation
1. Download or clone this repository.
2. Double-click **`setup.bat`**. 
3. *That's it.*

`setup.bat` silently downloads everything your system needs directly into the folder, builds the hidden execution layers, and registers LARS into your Windows User Logon registry. The daemon background watcher hides in your system memory, eagerly listening for the `ArmaReforger.exe` Window to boot. Once you play the game, your customized PyQt5 LARS Graphical Interface miraculously appears!
